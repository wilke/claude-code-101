# 2026-06-20 — Choosing the Elo K factor

Setting up the Elo pass over `matches.csv` before the tournament ratings feed the
predictor. The one free parameter that matters is `K`, the update step size.

- **Decision.** Use **K=40** for the finals. That's the value FIFA uses for World Cup
  matches in its own Elo-based ranking, and the football-ratings community treats it as
  standard for high-stakes tournament games (friendlies use a much smaller K). Bigger
  K = ratings respond faster to results, which feels right when a team's current form
  matters more than its two-year-old friendlies.
- **Home advantage.** Applied only when `neutral == False`. Every World Cup match is
  neutral, so inside the tournament no team gets a home bonus — the host nations
  included. (Carried over from the CLAUDE.md rule.)
- **Parameter.** `K = 40`, `home_adv = 65` Elo points (non-neutral only), base 1500.

**Result.** Ratings look sensible: the pre-tournament favourites (Brazil, France,
Argentina) sit at the top, which is what we'd expect.

**Note for later:** K=40 is the "standard" number, but standard-for-FIFA-rankings and
best-for-*our*-predictions aren't obviously the same thing. Worth a sensitivity check
once we have some tournament results to score against.
