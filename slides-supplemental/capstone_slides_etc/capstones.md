# Capstone Projects

BQPD-CPP
* Goal: Create a thread-safe C++ version of Roger Fletcher's F77 quadratic optimization solver 
* Outcome: C++ library implementing a robust QP solver, interfaced to UNO

CUTEst-C 
* Goal: Create a no-dependency C implementation of CUTEst (optimization test problem lib)
* Outcome: C library derived from a Fortran implementation for testing AD capabilities

ConDFO
* Goal: Develop and build a derivative-free optimization solver with constraints
* Outcome: Python implementation and research paper on constrained DFO

OptiUNO 
* Goal: Use openEvolve to search for an optimal optimization solver
* Outcome: Methodology for tuning parameters in optimization solver; maybe a paper.
